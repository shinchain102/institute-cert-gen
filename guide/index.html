Windows Setup

1. Install Python

  1. Download Python 3.x from python.org
  2. Run the installer
  3. Check "Add Python to PATH" during installation
  4. Verify installation: Open Command Prompt and run python --version

2. Install LibreOffice

  1. Download LibreOffice from libreoffice.org
  2. Run the installer with default options
  3. Add LibreOffice to system PATH:
    • Search for "Environment Variables" in Windows
    • Edit PATH variable
    • Add C:\Program Files\LibreOffice\program

3. Setup Application

  1. Clone or download the repository
  2. Open Command Prompt in the project directory
  3. Run: pip install -r requirements.txt
  4. Start the server: python main.py

Linux Setup

1. Install Python and LibreOffice

sudo apt update
sudo apt install python3 python3-pip libreoffice

2. Setup Application

# Clone repository
git clone [repository-url]
cd certificate-generator

# Install dependencies
pip3 install -r requirements.txt

# Run application
python3 main.py

Dependencies

Required Software

  • Python 3.x
  • LibreOffice (for PowerPoint conversion)
  • pip (Python package manager)

Python Packages

  • Flask - Web framework
  • python-docx - Word document processing
  • python-pptx - PowerPoint processing
  • pandas - Data file handling
  • docxtpl - Template processing
  • docx2pdf - PDF conversion

Troubleshooting

  1. Verify LibreOffice installation
  2. Check if LibreOffice is in system PATH
  3. Try running LibreOffice manually to confirm it works
  4. Check application logs for specific errors

  1. Ensure variables are in correct format: {{variable_name}}
  2. Check for typos and spacing
  3. Verify template file format is supported
